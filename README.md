# BioFuel-Hub
«BioFuel Hub» — это динамическое веб-приложение для производителей и покупателей топливных брикетов. Платформа позволяет производителям управлять каталогом своей продукции (оптом и в розницу), а покупателям — сравнивать характеристики, фильтровать предложения по цене и качеству, оставлять отзывы и формировать заказы напрямую.
